## Requirement
- Node >= v20.14.0 

## Install

Install the dependencies:

```bash
npm install
```

## Get Started

Start the dev server:

```bash
npm run dev
```

Build the app for production:

```bash
npm run build
```
Preview the production build locally:

```bash
npm run preview
```

Tests:

```bash
npm run test
```

Tests with coverage:

```bash
npm run test:coverage
```
